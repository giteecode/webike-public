## 二、所用技术

- #### 后台技术选型

  * Spring
  * SpringMVC
  * Mybatis
  * jsp
  * mysql

- #### 前端技术选型

  * EasyUI
  * jsp

## 六、安装教程

1. 使用Navicat或者其它工具、在mysql中创建对应名称的数据库、并导入项目的sql文件；
2. 使用IDEA/Eclipse导入项目，若为maven项目请选择maven，导入成功后请执行maven clean;maven install命令
3. 修改src\main\resources里面的jdbc.properties 里面的数据库配置
4. 将项目加入到tomcat容器里面，启动项目
